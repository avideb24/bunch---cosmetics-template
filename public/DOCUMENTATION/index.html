Thank you for purchasing the Bunch Cosmetics Template. This guide will help you set up and customize your template efficiently.

1. Installation

Ensure you have Node.js installed. Follow these steps:

npm install

To start the development server:

npm run dev

2. Folder Structure


      /src/
        ├── /app/             # Next.js pages
        ├── /components/      # UI components
        ├── /assets/          # Icon and images
        ├── /styles/          # Global CSS & Tailwind styles
        ├── /utils/           # Data / Helper functions
        ├── /config/          # Config data files
      /LICENSE.txt            # License file
      /README.md              # Quick setup guide
      /public/DOCUMENTATION/  # Full documentation

3. Customization

You can edit colors, font, site config and metadata in the following files:

  • /src/config/siteConfig.js
  • /src/config/metadataConfig.js
Change branding elements in /src/assets/. Change colors and font in /tailwind.config.mjs.

4. Deployment

To create a production build:

npm run build

To start the production server:

npm start
