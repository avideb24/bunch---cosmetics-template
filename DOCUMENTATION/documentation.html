1. Installation

Ensure you have Node.js installed. Then, follow these steps:

npm install

To start the development server:

npm run dev

2. Folder Structure

/public/ - Static assets
```

/src/ ├── /components/ - UI components ├── /pages/ - Next.js pages ├── /styles/ - CSS/Tailwind styles ├── /utils/ - Helper functions /LICENSE.txt - Licensing information /README.md - Quick setup guide /DOCUMENTATION/ - This documentation 

```

3. Customization


Edit site settings in /config/settings.js. Change branding elements in /public/.



4. Deployment


To create a production build:


npm run build


To start the production server:


npm start
